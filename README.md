# Callas Foundation Backend

## Overview
The Callas Foundation backend is an ASP.NET Core 8 Web API that provides RESTful endpoints for the Callas Foundation website.

## Technologies Used

- ASP.NET Core 8
- C#
- Entity Framework Core
- SQL Server LocalDB
- JWT Authentication
- Swagger/OpenAPI
- Postman
- Git & GitHub

## Features

- User Authentication (JWT)
- News Management
- Events Management
- Contact Messages
- Donations
- Volunteers
- Partners
- Gallery
- Resources
- Team Members
- Programmes

## Setup

1. Clone the repository.
2. Open the solution in Visual Studio 2022.
3. Restore NuGet packages.
4. Update the connection string if needed.
5. Run EF Core migrations.
6. Start the API.
7. Open Swagger.
